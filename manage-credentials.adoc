= 删除凭据
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
如果您不再需要一组凭据、可以从Workload Factory中将其删除。您只能删除与FSx for ONTAP文件系统无关的凭据。

.步骤
. 在Workload Factory控制台中，选择*Account*图标，然后选择*凭 据*。
. 在*凭据*页面上，选择一组凭据的操作菜单，然后选择*删除*。
. 选择*Remove*确认。
